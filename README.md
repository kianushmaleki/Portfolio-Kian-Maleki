# Kian Maleki — Portfolio

This is a minimal static portfolio page for Kian Maleki and includes the following sections:

- Header with name + navigation
- Hero section with a welcome message
- About Me
- Skills (data science skills)

Files:

- `index.html` — main HTML file
- `styles.css` — basic styling

This repository now includes a full CV section embedded in `index.html` (contact details, education, research, publications, teaching, and more).

How to view:

1. Open `index.html` in your web browser (double-click or open with your browser of choice).
2. Or start a simple file server from the folder (example using Python 3):

```powershell
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

Customize the name, email, phone number, and descriptions in `index.html` to make it your own. The site is pre-filled with the owner name and contact details for Kian Maleki.

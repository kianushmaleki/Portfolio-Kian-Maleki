Selected Projects

Beta Bank 2025

Repository

In this project, I developed a machine learning solution for Beta Bank to proactively address customer attrition, achieving a final F1-score of 0.62 and an AUC-ROC of 0.85, which successfully exceeded the target benchmark of 0.59. Leveraging a toolkit including Pandas, NumPy, and Scikit-Learn, I executed a rigorous pipeline that involved cleaning a 10,000-record dataset, engineering features through One-Hot Encoding and Standard Scaling, and resolving a significant 80/20 class imbalance using Upsampling and Balanced Class Weights. My exploratory data analysis using Seaborn revealed that age and product count were primary churn drivers, allowing me to refine a Random Forest Classifier that effectively balances precision and recall. This model provides the bank with a data-driven strategy to identify high-risk customers, allowing for targeted retention incentives that optimize marketing spend and protect long-term revenue.
